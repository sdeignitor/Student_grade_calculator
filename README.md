# 🎯 Student Grade Calculator

A simple Python project that calculates a student's grade based on marks entered by the user.

## 📌 Features

- Accepts student name and marks
- Validates input (0–100 only)
- Uses if-elif-else grading logic
- Displays encouraging messages
- Handles invalid inputs using a while loop
- Uses functions for clean and reusable code

---

## 🛠 Technologies Used

- Python 3
